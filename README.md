## After clone
1. ```cd paste-link```
2. ```composer install```
3. ```cp .env.example .env```
4. ```php artisan key:generate```
5. (database configuraton in .env file)
6. ```php artisan migrate --seed```
7. ```npm install```
8. ```npm run dev```
9. ```php artisan serve```
10. (default user in UserSeeder.php file)
